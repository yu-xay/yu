####PHP代码执行过程
1. Scanning(Lexing/Tokenize) 将php代码转换成语言片段(Tokens);单词符号
2. Parsing                   将Tokens转换成简单而有意义的表达式AST(抽象语法树)
3. Compilation               将抽象语法树编译成Opacode 操作码
4. Execution                 顺次执行Opacode，从而实现PHP脚本的功能
5. zend Vm                   Zend VM引擎 接收操作码 并执行
